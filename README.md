# Ncurses-Pong
A pong clone using ncurses

Make sure you have the ncurses library

To install use

    $ sudo apt-get install libncurses5-dev 
    
To compile use 

    $  make

To use

    $ ./pong

To use another color set (6 in total)

    $ ./pong -2 

    $ ./pong -3
